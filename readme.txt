# 🚗 Portafolio 3D de Modelos de Autos

Este proyecto es una aplicación web desarrollada con Next.js y Tailwind CSS que permite visualizar modelos 3D de autos en una galería tipo Pinterest. Incluye navegación dinámica, encabezados sticky, y una experiencia de usuario optimizada para dispositivos móviles y escritorio.

---

## 📦 Tecnologías utilizadas

- Frontend: Next.js, React, Tailwind CSS
- 3D:  Three.js, @react-three/fiber, @react-three/drei
- Backend Express: (Node.js)
- Base de datos: SQLite / PostgreSQL (según configuración)
- Estilo de código: ESLint + Prettier
